Module prelude

Source
Expand description

The crate prelude exports. Importing this with a wildcard will import most items from RIL that can be useful for image processing, along with bringing crucial traits into scope.

ยงExample

use ril::prelude::*;

// Prelude imported Image and Rgb
let image = Image::new(100, 100, Rgb::new(255, 0, 0));
// Prelude imported the Banded trait
let (r, g, b) = image.bands();
